# Fonio - v1.1

**Release Name**: Version 1.1
**Published**: 2020-09-17T14:54:53Z
**Repository**: https://github.com/foniod/foniod

---

This new version is a maintenance release which improves support for 5.7 or newer kernels, and particularly for Fedora, CentOS, and Red Hat based systems. `ingraind` now builds on the latest Rust releases, and requires LLVM 10 for building. In addition update dependencies to the latest versions.
