================================================================================

Compiles a Cairo project to Sierra.
Exits with 0/1 if the compilation succeeds/fails.

Usage: cairo-compile [OPTIONS] <PATH> [OUTPUT]

Arguments:
  <PATH>
          The Cairo project path

  [OUTPUT]
          The output file name (default: stdout)

Options:
  -s, --single-file
          Whether path is a single file

  -r, --replace-ids
          Replaces sierra ids with human-readable ones

  -i, --inlining-strategy <INLINING_STRATEGY>
          Overrides inlining behavior
          
          [default: default]

          Possible values:
          - default: Do not override inlining strategy
          - avoid:   Inline only in the case of an `inline(always)` annotation

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
